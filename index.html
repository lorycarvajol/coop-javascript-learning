✨ Introduction aux variables en JavaScript

1. Qu'est-ce qu'une variable ?

Une variable est comme une boîte dans laquelle on peut stocker une information. Par exemple :

let prenom = "Lory";

Ici, on crée une variable appelée prenom contenant le texte Lory.

2. Utiliser une variable

let age = 25;
let message = "Tu as " + age + " ans.";
alert(message);
