0. Introduction à JavaScript

1. Qu'est-ce que JavaScript ?

JavaScript est un langage de programmation qui permet d’ajouter de l’interactivité à une page web. Il complète le HTML (qui structure la page) et le CSS (qui la met en forme).

Grâce à JavaScript, on peut :

  • Réagir aux actions de l'utilisateur (clic, survol, saisie clavier...)
  • Modifier dynamiquement le contenu d'une page
  • Créer des animations, valider des formulaires, interagir avec un serveur, etc.

JavaScript est interprété directement dans le navigateur, ce qui permet d’avoir des effets en temps réel.

2. Où écrit-on du JavaScript ?

On peut écrire du JavaScript :

  • Directement dans la page HTML, entre des balises <script>
  • Dans un fichier externe (ex : script.js) que l'on relie dans le HTML via <script src="script.js"></script>
<script>
  alert('Bienvenue sur mon site !');
</script>

3. Comment ça fonctionne ?

Quand tu ouvres une page web, voici ce qui se passe :

  1. Le navigateur lit le HTML et construit la structure de la page.
  2. Il applique le CSS pour styliser les éléments.
  3. Enfin, il exécute le JavaScript pour rendre la page dynamique.

Le JavaScript interagit avec le DOM (Document Object Model), une représentation de la structure HTML sous forme d'objet manipulable avec le code.

1. Définition simple

Une variable est un conteneur qui permet de stocker une information, comme du texte, un nombre ou une valeur logique.

let nom = "Alice";

2. Pourquoi utiliser des variables ?

  • Stocker des informations à réutiliser
  • Faire des calculs
  • Afficher ou manipuler dynamiquement des données dans une page

3. Les types de variables

// Texte (chaîne de caractères)
let prenom = "Lory";

// Nombre
let age = 30;

// Booléen (vrai ou faux)
let estConnecte = true;

4. let, const ou var ?

  • let : valeur modifiable
  • const : valeur constante (non modifiable)
  • var : ancien mot-clé à éviter dans les nouveaux codes
const PI = 3.14;
let rayon = 5;
let surface = PI * rayon * rayon;
console.log(surface);
